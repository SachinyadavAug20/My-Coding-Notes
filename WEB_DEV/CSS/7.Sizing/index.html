Different sizing units in CSS

  • Px -pixels(not responsive)
  • vw -viewport width(% of width)
  • vh -viewport height(% of height)
  • em -*Current size(multiplier of current default or inherited size)
  • rem -*Current size of root element(root element are html)
  • vmin & vmax -this is same as vw/vh it decides on min and max of hight/width
  • % -it is percentage of parent value(parents value must be set before hand)
